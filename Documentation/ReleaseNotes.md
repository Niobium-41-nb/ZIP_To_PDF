## 🚀 部署说明

### 系统要求
- **操作系统**: Windows 7/8/10/11 (64位)
- **内存**: 4GB RAM (推荐)
- **磁盘空间**: 2GB 可用空间
- **网络**: 需要网络连接（JM漫画下载功能）

### 安装步骤
1. 下载 `压缩包转PDF工具.exe` 文件
2. 将文件放在合适的目录（如桌面或专用文件夹）
3. 双击运行或通过命令行使用
4. 无需安装Python或其他依赖

### 运行方式
```cmd
# 方法1: 双击运行（交互模式）
压缩包转PDF工具.exe

# 方法2: 命令行启动Web界面
压缩包转PDF工具.exe --web

# 方法3: 命令行下载JM漫画
压缩包转PDF工具.exe <漫画ID>
```
