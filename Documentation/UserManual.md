# 压缩包转PDF工具 - 可执行文件使用说明

## 📦 文件信息
- **文件名**: `压缩包转PDF工具.exe`
- **版本**: 1.0
- **平台**: Windows
- **类型**: 单文件可执行程序
- **大小**: 约 50-100MB (取决于包含的依赖)

## 🚀 快速开始

### 方法1: 交互模式（推荐新手）
```cmd
压缩包转PDF工具.exe
```
然后根据菜单选择功能：
1. 启动Web界面
2. 下载JM漫画并转换为PDF  
3. 清理下载文件
4. 清理临时文件
5. 清理temp文件夹中的所有内容

### 方法2: Web界面模式
```cmd
压缩包转PDF工具.exe --web
```
访问 http://localhost:5000 使用图形界面

### 方法3: 命令行下载JM漫画
```cmd
压缩包转PDF工具.exe <JM漫画ID>
```
示例：
```cmd
压缩包转PDF工具.exe 422866
```

### 方法4: 清理功能
```cmd
# 清理下载文件
压缩包转PDF工具.exe --cleanup

# 清理临时文件
压缩包转PDF工具.exe --cleanup-temp

# 清理temp文件夹中的所有内容
压缩包转PDF工具.exe --cleanup-all-temp
```

## 🎯 功能特性

### Web界面功能
- ✅ **拖拽上传**: 支持ZIP, RAR, 7Z等压缩格式
- ✅ **实时进度**: 显示上传、解压、处理、生成全过程
- ✅ **多种下载**: 支持完整包下载和单个PDF下载
- ✅ **现代化界面**: 响应式设计，操作简单

### 命令行功能
- ✅ **JM漫画下载**: 直接输入ID自动下载并转换
- ✅ **批量处理**: 支持多个漫画ID
- ✅ **自动清理**: 处理完成后自动清理临时文件
- ✅ **进度显示**: 实时显示下载和处理进度

### 系统管理
- ✅ **文件清理**: 多种清理选项
- ✅ **大文件支持**: 最大支持1GB文件
- ✅ **错误恢复**: 完善的错误处理机制

## 📁 文件结构

运行程序时会自动创建以下目录：
```
项目目录/
├── uploads/          # 上传文件临时存储
├── temp/             # 解压临时目录  
├── outputs/          # 生成的PDF文件
├── download/         # JM漫画下载目录
└── 压缩包转PDF工具.exe  # 主程序
```

## 🔧 使用示例

### 示例1: 使用Web界面处理压缩包
1. 运行: `压缩包转PDF工具.exe --web`
2. 打开浏览器访问: http://localhost:5000
3. 拖拽上传ZIP/RAR文件
4. 等待处理完成
5. 下载生成的PDF文件

### 示例2: 命令行下载JM漫画
```cmd
压缩包转PDF工具.exe 422866
```
程序将：
1. 自动下载JM漫画422866
2. 转换为PDF格式
3. 显示处理进度
4. 完成后显示文件位置

### 示例3: 批量清理
```cmd
# 清理所有临时文件
压缩包转PDF工具.exe --cleanup-all-temp

# 清理下载的漫画文件
压缩包转PDF工具.exe --cleanup
```

## ⚠️ 注意事项

### 系统要求
- **操作系统**: Windows 7/8/10/11
- **内存**: 建议4GB以上
- **磁盘空间**: 至少2GB空闲空间
- **网络**: 稳定的网络连接（JM漫画下载需要）

### 使用限制
- 单个文件最大1GB
- 支持常见图片格式: JPG, PNG, WEBP, GIF, BMP
- 支持压缩格式: ZIP, RAR, 7Z, TAR, GZ, BZ2

### 常见问题

**Q: 程序无法启动？**
A: 确保系统已安装必要的运行库，可以尝试安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

**Q: Web界面无法访问？**
A: 检查防火墙设置，确保5000端口未被占用

**Q: 下载JM漫画失败？**
A: 检查网络连接，确认JM漫画ID正确

**Q: 磁盘空间不足？**
A: 使用清理功能释放空间：`压缩包转PDF工具.exe --cleanup-all-temp`
